# E-Commerce Sales Analysis & Profit Prediction

## 📌 Project Overview
This project analyzes an e-commerce sales dataset to understand key factors affecting sales performance and profitability. The analysis includes data cleaning, exploratory data analysis (EDA), feature engineering, and machine learning models to predict whether an order will generate profit or loss.

The project also includes an **interactive Power BI dashboard** to visualize business insights such as sales trends, regional performance, product performance, and discount impact.

---

## 🎯 Objectives
- Analyze sales and profitability patterns in e-commerce transactions.
- Identify key factors affecting profit and loss.
- Build machine learning models to predict order profitability.
- Create a Power BI dashboard for business intelligence and visualization.

---

## 📊 Dataset
The dataset contains information about **9,700+ e-commerce transactions** including:

- Order ID  
- Product Category & Sub-category  
- Sales  
- Profit  
- Discount  
- Shipping Cost  
- Customer Segment  
- Region  
- Order Date & Ship Date  

---

## 🔍 Exploratory Data Analysis (EDA)
EDA was performed to identify trends and relationships in the dataset. Key analyses include:

- Sales distribution analysis  
- Discount vs Profit relationship  
- Sales by product category  
- Regional sales performance  
- Monthly sales trends  
- Correlation analysis between numerical features  

---

## 🤖 Machine Learning Models
Three classification models were built to predict whether an order will generate **profit or loss**:

- Logistic Regression  
- Decision Tree  
- Random Forest  

### Model Performance

| Model | Accuracy |
|------|----------|
| Logistic Regression | **92.11%** |
| Decision Tree | **88.11%** |
| Random Forest | **91.93%** |

Models were evaluated using:

- Accuracy  
- Precision  
- Recall  
- F1 Score  
- Confusion Matrix  
- ROC Curve  

---

## 📈 Key Insights
- Higher discounts tend to reduce overall profitability.
- The **Technology category** contributes significantly to total sales.
- Some regions generate higher revenue than others.
- A small number of products contribute to a large portion of total sales.
